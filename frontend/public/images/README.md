# Pasta de Imagens

## Como adicionar a logo da empresa:

1. Coloque o arquivo da logo nesta pasta (`frontend/public/images/`)
2. Nomeie o arquivo como `logo.png` (ou `logo.svg`, `logo.jpg`, etc)
3. Se usar outro nome, edite o arquivo `frontend/src/app/app/layout/header/header.html` 
   e altere o caminho na tag `<img src="/images/logo.png">`

### Formatos suportados:
- PNG (recomendado para logos com transparÃªncia)
- SVG (recomendado para qualidade vetorial)
- JPG/JPEG
- WEBP

### Tamanhos recomendados:
- Logo para header: altura de 40px (largura proporcional)
- Logo para hero: altura de 200px (largura proporcional)
