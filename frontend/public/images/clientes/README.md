# Logos dos Clientes

Esta pasta contém as logos das empresas clientes da AR Contabilidade.

## Estrutura de Nomenclatura

As logos devem ser nomeadas usando o CNPJ da empresa (apenas números, sem formatação):

- Exemplo: `33236630000119.png` (para CNPJ 33.236.630/0001-19)

## Formato Recomendado

- **Formato**: PNG ou SVG (preferencialmente PNG com fundo transparente)
- **Tamanho**: Mínimo 200x200px, máximo 800x800px
- **Resolução**: 72-150 DPI (adequado para web)
- **Fundo**: Transparente ou branco

## Lista de Clientes

Consulte o plano de sprints para a lista completa de clientes com seus respectivos CNPJs.

## Placeholder

Se uma logo não for encontrada, o sistema utilizará um placeholder genérico.
